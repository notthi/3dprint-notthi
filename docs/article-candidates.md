## 優先度の凡例

| 優先度 | 基準 |
|--------|------|
| **S** | 高検索量 × 高収益ポテンシャル。今すぐ書くべき記事 |
| **A** | 検索需要が高くロングテールにも強い。次のバッチで対応 |
| **B** | 内部リンク強化・ロングテール狙い。コンテンツ充実期に対応 |

## 新記事追加時のチェックリスト

- [ ] `content/articles/_template.mdx` をコピーして記事を執筆
- [ ] `content/articles/_template.meta.ts` をコピーしてメタデータを記入
- [ ] `src/lib/articles.ts` に `ArticleEntry` を追加
- [ ] `src/app/{カテゴリ}/{スラッグ}/page.tsx` を作成（thin wrapper）
- [ ] `npm run build` でビルド確認
- [ ] `git push` → Vercel自動デプロイ
