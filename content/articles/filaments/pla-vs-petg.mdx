<section id="pla" style={S.section}>
  <h2 style={S.h2}>PLAの特徴</h2>
  <div style={S.p}>
    PLA（ポリ乳酸）は<strong style={{ color: "#10b981" }}>初心者に最もおすすめのフィラメント</strong>です。印刷温度は190〜220℃と低めで、ベッド温度は不要か低温（40〜60℃）でOK。反りにくく、ほとんどの機種でそのまま印刷できます。
  </div>
  <div style={S.p}>
    色のバリエーションが豊富で、マット・シルク・グリッターなど仕上がりの種類も多いのが魅力です。
  </div>
  <div style={S.tip}>
    <div style={{ color: "#10b981", fontWeight: "700", marginBottom: "0.5rem" }}>✅ PLAが向いている用途</div>
    <div style={{ ...S.p, marginBottom: 0 }}>
      フィギュア・インテリア雑貨などの見た目重視の造形 / 試作品・プロトタイプ / 室内で使う小物（スタンド、ホルダー、整理グッズ）
    </div>
  </div>
  <div style={S.p}>
    ただし耐熱性が低いため、<strong style={{ color: "#f1f5f9" }}>夏の車内への放置・直射日光が当たる場所への設置は変形のリスクがあります</strong>。屋外や食器への使用も向きません。
  </div>
</section>

<section id="petg" style={S.section}>
  <h2 style={S.h2}>PETGの特徴</h2>
  <div style={S.p}>
    PETG（ポリエチレンテレフタレートグリコール）は<strong style={{ color: "#06b6d4" }}>PLAの扱いやすさとABSの強度を兼ね備えた素材</strong>です。強度・耐熱性・耐水性がPLAより優れており、機能パーツの製作に向いています。
  </div>
  <div style={S.p}>
    印刷温度は230〜250℃、ベッド温度は70〜90℃が目安です。
  </div>
  <div style={S.tip}>
    <div style={{ color: "#06b6d4", fontWeight: "700", marginBottom: "0.5rem" }}>✅ PETGが向いている用途</div>
    <div style={{ ...S.p, marginBottom: 0 }}>
      ドローンのフレームやパーツなど強度が必要なもの / 水まわりで使うもの / 屋外で使うもの / 機械部品・工具補助パーツ
    </div>
  </div>
  <div style={S.p}>
    難点は<strong style={{ color: "#f1f5f9" }}>糸引き（ストリング）が出やすい</strong>こと。リトラクション設定を調整する手間がかかります。また表面がPLAより少しザラつきやすいため、見た目重視のモデルには向きません。
  </div>
</section>
