<section id="pla" style={S.section}>
  <h2 style={S.h2}>PLA — 最初の1択</h2>
  <div style={S.p}>
    3Dプリンター初心者には<strong style={{ color: "#10b981" }}>無条件でPLAを推奨</strong>します。印刷のしやすさ、コストパフォーマンス、仕上がりの美しさがバランスよく揃った素材です。植物由来（トウモロコシのデンプン等）の生分解性プラスチックで、印刷時の嫌な臭いも少ないです。
  </div>
  <div style={S.p}>
    <strong style={{ color: "#f1f5f9" }}>弱点</strong>は耐熱性の低さ。夏の車内（80℃超）に放置すると変形します。屋外での長期使用や熱がかかる用途には向きません。
  </div>
</section>

<section id="pla-plus" style={S.section}>
  <h2 style={S.h2}>PLA+ — PLAの強化版</h2>
  <div style={S.p}>
    同じ印刷しやすさを維持しながら衝撃強度を高めたグレードです。Bambu Lab Pure PLA、eSUN PLA+、Polymaker PolyMax PLAなどが代表的。PLA対比で衝撃強度が約1.5〜2倍になります。最初からPLA+を選ぶのも合理的な選択です。
  </div>
</section>

<section id="petg" style={S.section}>
  <h2 style={S.h2}>PETG — 実用品の定番</h2>
  <div style={S.p}>
    食品容器にも使われるPETGは<strong style={{ color: "#06b6d4" }}>安全性が高く、強度も十分</strong>な優等生素材です。PLAより耐熱・耐衝撃に優れ、実用品を作るならまずPETGを検討してください。
  </div>
  <div style={S.p}>
    欠点は<strong style={{ color: "#f1f5f9" }}>糸引き（ストリング）</strong>が出やすいこと。リトラクション設定を丁寧に調整する必要があります。
  </div>
</section>

<section id="abs" style={S.section}>
  <h2 style={S.h2}>ABS — 上級者向け</h2>
  <div style={S.p}>
    古くから使われる素材ですが、<strong style={{ color: "#f1f5f9" }}>反り（ワーピング）</strong>が非常に起きやすく、温度管理が難しい。エンクロージャー（密閉チャンバー）があるプリンター（Bambu Lab P2S等）でないと安定した印刷は難しいです。
  </div>
  <div style={S.warn}>
    <div style={{ color: "#ef4444", fontWeight: "700", marginBottom: "0.5rem" }}>⚠️ 初心者はABSを避けてください</div>
    <div style={{ ...S.p, marginBottom: 0 }}>
      ABSが必要な耐熱用途には、印刷しやすい<strong style={{ color: "#f1f5f9" }}>ASA</strong>で代替できることが多いです。
    </div>
  </div>
</section>

<section id="asa" style={S.section}>
  <h2 style={S.h2}>ASA — 屋外最強素材</h2>
  <div style={S.p}>
    ABSの欠点（UV劣化）を改善した素材です。<strong style={{ color: "#f1f5f9" }}>紫外線や雨風に強く</strong>、屋外での長期使用に最適です。庭のパーツ、車外装備、ドローンのフレーム等に向いています。印刷の難しさはABSと同等ですが、ABSよりも積極的に選ぶ価値があります。
  </div>
</section>

<section id="tpu" style={S.section}>
  <h2 style={S.h2}>TPU — 柔軟素材の王様</h2>
  <div style={S.p}>
    ゴムのような弾性を持ち、<strong style={{ color: "#8b5cf6" }}>曲げても割れない</strong>のが最大の特徴です。スマートフォンケース、釣りのルアー、ラジコンタイヤ、靴底パーツ等に使われます。
  </div>
  <div style={S.p}>
    柔らかいがゆえに送り出し（エクストルーダー）で詰まりやすいため、印刷速度を遅めに設定する必要があります。Bambu Labのような高速プリンターでも、TPUは低速モードで印刷するのが基本です。
  </div>
</section>

<section id="nylon" style={S.section}>
  <h2 style={S.h2}>Nylon / PC — 上級エンジニアリング素材</h2>
  <div style={S.p}>
    <strong style={{ color: "#f1f5f9" }}>Nylon（ナイロン）</strong>は機械部品レベルの強度を持つ上級者向け素材です。吸湿性が非常に高いため、乾燥保管が必須。フィラメントドライヤーを使いながら印刷しないと品質が安定しません。
  </div>
  <div style={S.p}>
    <strong style={{ color: "#f1f5f9" }}>PC（ポリカーボネート）</strong>は最高クラスの強度・耐熱性を誇りますが、260〜300℃という高い印刷温度が必要で、対応プリンターが限られます。真剣に工業用途を考えている方向けです。
  </div>
</section>
