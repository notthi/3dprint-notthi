<Lead>
PLAに慣れてきたら次に試したいのが**PETG**です。PLAより強度・耐熱性・耐水性が高く、実用パーツ制作に向いています。

ただし「PETGは糸引きが出やすい」「設定が難しい」という声もあります。この記事では**実際に使って品質が安定していたPETG 5選**を紹介します。
</Lead>

## PETGを選ぶ3つの基準

<Steps items={[
  { title: "糸引き（ストリング）の少なさ", body: "PETGは粘度が高く糸引きが出やすい素材。品質の良いフィラメントはこれが抑えられており、後処理が楽になります。" },
  { title: "寸法精度（±0.02mm以内が目安）", body: "精度が低いとノズル詰まりの原因になります。機能パーツ・精密部品ほど精度の高いブランドを選ぶことが重要。" },
  { title: "乾燥状態の管理", body: "PETGはPLAより吸湿しやすい。湿気を吸ったまま印刷すると気泡・ブツブツが出ます。密封パッケージ・乾燥剤入りのものを選びましょう。" },
]} />

## PETGフィラメントおすすめ5選

### 1. eSUN PETG ― 定番の安心感

印刷温度：230〜240℃ / ベッド：80℃

PETGの定番ブランド。品質安定・糸引き少なめ。どのプリンターでも安定して使えます。初めてPETGを試すならまずこれ。

<ProductCard productId="esun-petg" />

### 2. Bambu Lab PETG HF ― Bambu機種との相性◎

印刷温度：240℃ / ベッド：70℃

Bambu Lab純正。Bambu Studioにプリセット入りで設定不要。A1・P1S・X1Cユーザーに最も手軽な選択肢。

<ProductCard productId="bambu-petg-hf" />

### 3. Polymaker PolyLite PETG ― 高品質路線

印刷温度：230〜245℃ / ベッド：75〜85℃

寸法精度が高く機械パーツ・精度が求められる用途向き。表面仕上げもきれいで品質重視派に。

<ProductCard productId="polymaker-polylite-petg" />

### 4. SUNLU PETG ― コスパ重視

印刷温度：230〜250℃ / ベッド：80℃

最安値クラス。試作品・日用品・プロトタイプ向き。大量印刷や複数色を試したい場合に。ロットによってばらつきあり。

<ProductCard productId="sunlu-petg" />

### 5. Creality Hyper PETG ― 高速印刷向け

印刷温度：240〜260℃ / ベッド：80℃

高速印刷（300mm/s以上）対応PETG。Creality K1・Bambu Lab高速機種ユーザーに向いています。

<ProductCard productId="creality-hyper-petg" />

## PETGの印刷設定のコツ

PLAから乗り換えた際につまずきやすいポイントをまとめます。

<Steps items={[
  { title: "ベッド温度は80℃前後に設定する", body: "低すぎると剥がれ、高すぎるとビルドプレートに貼り付きすぎます。PEI・磁気フレックスシートとの組み合わせが定着しやすい。" },
  { title: "リトラクションを調整する", body: "糸引きが出る場合はリトラクション距離を0.5〜1mm増やしてみましょう。増やしすぎるとノズル詰まりの原因になります。" },
  { title: "印刷速度は最初ゆっくりから", body: "50〜60mm/s程度で試して、問題なければ上げていくのが失敗しにくいアプローチです。" },
  { title: "湿気対策を忘れずに", body: "使い終わったフィラメントは乾燥剤と一緒にジップロックで保管しましょう。フィラメント乾燥機があればより安心です。" },
]} />
