<Lead>
**ノズル詰まり（クロッグ）は3Dプリンターで最もよく起こるトラブルのひとつです。** フィラメントが出なくなったり、層がスカスカになったりした場合は、まずノズル詰まりを疑いましょう。

ほとんどのケースは**コールドプル**で解決できます。交換が必要な場合も手順は簡単です。
</Lead>

## 主な原因

| 原因 | 詳細 |
|---|---|
| フィラメントの吸湿 | 水分を含んだフィラメントは気泡が発生し、ノズル内で炭化して詰まりの原因になる |
| 低すぎる温度での印刷 | フィラメントが十分に溶けずノズル内で固まる |
| フィラメントの炭化 | 長時間の印刷や高温放置でフィラメントが焦げて詰まる |
| 異なる素材の混在 | 前の素材が残ったまま新しい素材を入れると詰まりやすい |
| ノズルの摩耗・劣化 | カーボンファイバー入りフィラメントなどによる内壁の削れ |

## 解決①：コールドプル（最も効果的）

コールドプルはノズル内の汚れや詰まりを引き抜く方法です。ほとんどの詰まりはこれで解消できます。

<Steps items={[
  { title: "ノズルを印刷温度（PLAなら220℃）まで加熱する" },
  { title: "フィラメントを少し押し出して、ノズル内を満たす" },
  { title: "温度をゆっくり下げ、PLA: 70〜90℃ / PETG: 90〜110℃ まで冷ます" },
  { title: "フィラメントをゆっくり、一定の力で引き抜く（急いで引かない）" },
  { title: "引き抜いたフィラメントの先端が汚れている間は1〜3を繰り返す" },
]} />

<Tip title="成功のサイン">
引き抜いたフィラメントの先端がノズル形状に成形されていれば成功です。先端がきれいになるまで繰り返してください。
</Tip>

## 解決②：温度を上げて押し出す

コールドプルで改善しない場合は、通常より**10〜20℃高い温度**に設定してフィラメントを押し出してみましょう。

例：PLAを通常200℃で印刷している場合 → 220〜230℃に上げて、少量押し出す。固まった樹脂が溶けて出てくることがあります。

<Warn title="注意">
素材ごとの最大温度を超えないようにしてください（PLA: 240℃、PETG: 260℃が目安）。過熱するとフィラメントが焦げて悪化することがあります。
</Warn>

## 解決③：ノズル交換

コールドプルと高温押し出しを試しても改善しない場合は、ノズル交換が最も確実な解決策です。

ノズルは消耗品です。詰まりが繰り返す場合も迷わず交換しましょう。交換用ノズルは数百円で入手できます。

| 使用素材 | 交換目安 | 推奨ノズル |
|---|---|---|
| PLA / PETG | 3〜6ヶ月（使用頻度による） | 真鍮ノズル |
| PETG / ABS混用 | 1〜3ヶ月 | 真鍮ノズル |
| カーボンファイバー入り | 50〜100時間 | 硬化鋼ノズル（必須） |

## 詰まりを防ぐ予防策

<Tip title="詰まらせないための3つの習慣">
**① フィラメントを乾燥させる** — 湿気を吸ったフィラメントは詰まりの最大原因。未使用時はジップロック＋乾燥剤で保管、印刷前に乾燥機で乾燥させる。

**② 素材交換時にパージする** — 異なる素材に変える際は、旧素材が完全に出るまでパージ（押し出し）する。

**③ 月1回のコールドプル** — 詰まっていなくても定期的にコールドプルすると、蓄積した汚れを取り除けます。
</Tip>

## まとめ

### ノズル詰まり対処フロー

**Step 1：** まずコールドプルを試す（2〜3回繰り返す）

**Step 2：** 改善しなければ高温で押し出す

**Step 3：** それでもダメならノズル交換（数百円で解決）

**→ 予防が最大の対策。フィラメントの乾燥と定期メンテを忘れずに。**
