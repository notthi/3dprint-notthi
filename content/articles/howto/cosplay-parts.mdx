<section id="finish" style={S.section}>
  <h2 style={S.h2}>仕上げ・塗装の流れ</h2>
  <div style={S.p}>
    印刷後の仕上げ作業がクオリティを大きく左右します。基本の4ステップを押さえましょう。
  </div>
  <div style={S.card}>
    <div style={{ ...S.p, marginBottom: "0.75rem" }}>
      <strong style={{ color: "#f1f5f9" }}>Step 1：やすりがけ</strong><br />
      <span style={{ color: "#94a3b8" }}>120番→240番→400番の順で積層痕を削る。水研ぎペーパーを使うとより滑らかに。</span>
    </div>
    <div style={{ ...S.p, marginBottom: "0.75rem" }}>
      <strong style={{ color: "#f1f5f9" }}>Step 2：サーフェイサー塗布</strong><br />
      <span style={{ color: "#94a3b8" }}>グレーのサーフェイサー（サフ）を薄く2〜3回吹く。積層痕・傷が可視化されて修正しやすくなる。</span>
    </div>
    <div style={{ ...S.p, marginBottom: "0.75rem" }}>
      <strong style={{ color: "#f1f5f9" }}>Step 3：塗装</strong><br />
      <span style={{ color: "#94a3b8" }}>ラッカー系スプレー塗料またはエアブラシで塗装。下地色（黒や白）→本体色の順で塗ると発色が良い。</span>
    </div>
    <div style={{ ...S.p, marginBottom: 0 }}>
      <strong style={{ color: "#f1f5f9" }}>Step 4：コーティング</strong><br />
      <span style={{ color: "#94a3b8" }}>つや消し・半光沢・光沢クリアーでコーティング。擦れ・剥がれを防ぎ、仕上がりに統一感が出る。</span>
    </div>
  </div>
</section>
