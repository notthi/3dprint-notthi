<Lead>
**Thingiverseは世界最大の無料3Dモデル共有サイトで、数百万点のSTLデータが無料でダウンロードできます。**

アカウント登録（無料）するだけで使え、初心者から上級者まで幅広いユーザーが愛用しています。Bambu Lab機種には**Makerworld**もおすすめですが、データ量と多様性でThingiverseは今も最大規模です。
</Lead>

## 基本的な使い方

<Steps items={[
  { title: "thingiverse.comにアクセスしてアカウント登録（無料・メールアドレスのみ）" },
  { title: "英語で検索する（日本語は結果が少ない）", body: "例：「phone stand」「cable clip」「vase」「fidget cube」「3DBenchy」" },
  { title: "「Most Downloaded（多くダウンロードされた順）」でソートして品質の高いものを選ぶ" },
  { title: "プレビュー画像・コメント・ライセンスを確認" },
  { title: "「Download All Files」ボタンでSTLファイルをダウンロード" },
]} />

ダウンロードしたSTLファイルはスライサー（Bambu Studio・OrcaSlicerなど）に取り込んで印刷設定を行います。

## 品質の高いモデルの見つけ方

<Tip title="💡 選び方の3つのポイント">
**① ダウンロード数10,000以上を目安にする** — 多くの人が使っているモデルは品質が高い可能性が高い

**② 実際の印刷写真があるモデルを選ぶ** — 3Dレンダリングのみのモデルは実際に印刷できない場合がある

**③ ネガティブコメントが多いものは避ける** — 「印刷できない」「寸法が違う」というコメントは要注意
</Tip>

## ライセンスに注意

個人利用・非営利目的であればほとんどのモデルは自由に使えますが、販売・商用利用を考える場合はCreative Commonsライセンスを必ず確認しましょう。

| ライセンス | 商用利用 | 改変 | 注意 |
|---|---|---|---|
| CC BY | ○ | ○ | クレジット表記必要 |
| CC BY-NC | ✗ | ○ | 非商用のみ |
| CC BY-SA | ○ | ○ | 同一ライセンスで配布 |
| CC BY-NC-SA | ✗ | ○ | 非商用・同一ライセンス |

## 代替サイト比較

| サイト | 特徴 | おすすめ |
|---|---|---|
| Printables | Prusa運営。データ品質が高くモデレートされている | 品質重視の方 |
| Makerworld | Bambu Lab運営。Bambu Studioとの連携がスムーズ | Bambu Lab機種ユーザー |
| MyMiniFactory | フィギュア・ゲーム系のデータが豊富。有料モデルも多い | フィギュア・ゲーム好き |
| Thingiverse | データ量が最多。品質にばらつきあり | 多様なジャンルを探す |

## 初心者がまず印刷するおすすめモデル

**3DBenchy**

キャリブレーション用の小型ボート。世界標準のベンチマークモデルで、印刷品質のチェックに最適です。検索：「3DBenchy」

**Flexi Rex**

関節が動く恐竜。PLAで印刷できる柔軟造形の定番で、完成したときの満足感が高いです。検索：「Flexi Rex」

**Voronoi Vase**

幾何学的な花瓶。見栄えが良く、花瓶モード（螺旋印刷）で高速印刷できます。検索：「Voronoi Vase」

## まとめ

**Bambu Lab機種ユーザー** → Makerworldが使いやすさ◎

**多様なデータを探したい** → Thingiverseのデータ量は最多

**品質重視** → Printablesがモデレーションされていて安心

→ まずMakerworldかThingiverseで「3DBenchy」を検索して印刷してみましょう。
