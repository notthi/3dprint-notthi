<Lead>
**3Dプリンターを手に入れたら、まずフリーのSTLデータを印刷してみましょう。** 数百万点以上のモデルが無料・有料で公開されており、自分でモデリングしなくても多くのものを印刷できます。
</Lead>

## Thingiverse（世界最大・老舗）

**URL: thingiverse.com**

Makerbot（現在はDiginamo）が運営する世界最大のSTLデータ共有サイト。登録モデル数は600万点以上で、どんなジャンルも検索すれば見つかります。すべて無料でダウンロード可能。

<SpecList items={[
  { label: "メリット", value: "データ数が圧倒的・完全無料・検索しやすい" },
  { label: "デメリット", value: "品質にばらつきあり・サイトが重くなることがある・古いデータが多い" },
  { label: "こんな人に", value: "まず試したい初心者・とにかく数が必要な人" },
]} />

## Printables（高品質・Prusa公式）

**URL: printables.com**

Prusa Researchが運営するSTL共有プラットフォーム。品質審査が厳しく、高品質なモデルが揃っています。印刷に適したモデルかどうか評価がついており、失敗リスクが低いです。

<SpecList items={[
  { label: "メリット", value: "品質が高い・コミュニティが活発・ポイント制でデザイナーに還元" },
  { label: "デメリット", value: "Thingiverseに比べてデータ数は少ない" },
  { label: "こんな人に", value: "品質重視・失敗したくない・コミュニティ参加したい人" },
]} />

## MakerWorld（Bambu Lab公式）

**URL: makerworld.com**

Bambu Labが運営するプラットフォーム。Bambu製品ユーザーが特に使いやすく、プリントプロファイルが自動でBambu Studioに読み込まれます。急速に成長中で最新モデルが多いです。

<SpecList items={[
  { label: "メリット", value: "Bambu Studioと直結・プロファイル自動設定・急速に増えるモデル数" },
  { label: "デメリット", value: "Bambu機種以外では恩恵が薄い・まだデータ数がやや少ない" },
  { label: "こんな人に", value: "Bambu Lab機種ユーザー（A1 mini・P2S等）" },
]} />

## Cults3D（デザイナー系・有料多め）

**URL: cults3d.com**

フランス発のデザイナー向けプラットフォーム。無料モデルもありますが、有料・高品質なオリジナルデザインが充実しています。コスプレ・フィギュア・インテリア系が特に豊富です。

<SpecList items={[
  { label: "メリット", value: "デザインの質が高い・コスプレ・フィギュア系が充実・デザイナー直接購入" },
  { label: "デメリット", value: "良いモデルは有料（$1〜$30程度）・英語・フランス語が多い" },
  { label: "こんな人に", value: "高品質なデザインを購入したい・コスプレ・プロップ制作者" },
]} />

## MyMiniFactory（コミュニティ品質保証）

**URL: myminifactory.com**

実際に印刷テストされたモデルのみを掲載する方針で、品質が保証されています。無料・有料混在ですが、ゲームフィギュア・ミニチュア（卓上RPG用）系が特に充実しています。

<SpecList items={[
  { label: "メリット", value: "テスト済みで印刷失敗が少ない・ミニチュア・ゲームフィギュアに強い" },
  { label: "デメリット", value: "ジャンルの偏りあり・有料モデルが多い" },
  { label: "こんな人に", value: "卓上RPG・ミニチュア・ゲームフィギュア愛好家" },
]} />

## 比較まとめ

| サイト | データ数 | 料金 | 品質 | 特徴 |
|---|---|---|---|---|
| Thingiverse | 600万+ | 無料 | ばらつきあり | とにかく多い |
| Printables | 100万+ | 無料 | 高め | Prusa公式・品質安定 |
| MakerWorld | 急増中 | 無料中心 | 中〜高 | Bambu連携が最強 |
| Cults3D | 40万+ | 無料〜有料 | 高い | デザイナー系・コスプレ |
| MyMiniFactory | 40万+ | 無料〜有料 | テスト済み | ミニチュア・RPG |
