<section id="checklist" style={S.section}>
  <h2 style={S.h2}>反り対策チェックリスト（優先順）</h2>
  <div style={S.step}>
    <span style={S.num}>①</span>
    <strong style={{ color: "#f1f5f9" }}>ベッドを清掃する（最初に試すべき）</strong>
    <div style={{ ...S.muted, marginTop: "0.25rem" }}>IPA・無水エタノールで油分を拭き取る。指紋の油分だけで定着が大幅に落ちる</div>
  </div>
  <div style={S.step}>
    <span style={S.num}>②</span>
    <strong style={{ color: "#f1f5f9" }}>ベッド温度を上げる</strong>
    <div style={{ ...S.muted, marginTop: "0.25rem" }}>PLA→55〜65℃、PETG→75〜85℃、ABS→100〜110℃</div>
  </div>
  <div style={S.step}>
    <span style={S.num}>③</span>
    <strong style={{ color: "#f1f5f9" }}>スティックのりを塗る</strong>
    <div style={{ ...S.muted, marginTop: "0.25rem" }}>Prittなどの水性のりをビルドプレートに薄く塗る。コストほぼゼロで効果的</div>
  </div>
  <div style={S.step}>
    <span style={S.num}>④</span>
    <strong style={{ color: "#f1f5f9" }}>ブリムを追加する</strong>
    <div style={{ ...S.muted, marginTop: "0.25rem" }}>スライサーでブリム（5〜10mm幅）を追加して定着面積を増やす。最も確実な対策のひとつ</div>
  </div>
  <div style={S.step}>
    <span style={S.num}>⑤</span>
    <strong style={{ color: "#f1f5f9" }}>Zオフセットを調整する</strong>
    <div style={{ ...S.muted, marginTop: "0.25rem" }}>ノズルとベッドの距離を微調整。近すぎても遠すぎてもNG。第一層がしっかり潰れる距離に</div>
  </div>
  <div style={S.step}>
    <span style={S.num}>⑥</span>
    <strong style={{ color: "#f1f5f9" }}>扇風機・エアコンの風が当たらない場所に移動する</strong>
    <div style={{ ...S.muted, marginTop: "0.25rem" }}>外部からの冷気が急激な温度変化を引き起こす。印刷中の冷風は反りの大敵</div>
  </div>
  <div style={S.step}>
    <span style={S.num}>⑦</span>
    <strong style={{ color: "#f1f5f9" }}>密閉チャンバーを使う（ABS・ASAのみ）</strong>
    <div style={{ ...S.muted, marginTop: "0.25rem" }}>ABS・ASAは密閉チャンバー搭載機（Bambu Lab P2S等）での使用が前提</div>
  </div>
</section>
