<section id="bambu" style={S.section}>
  <h2 style={S.h2}>Bambu Lab機種での手順</h2>
  <div style={S.p}>
    Bambu Lab機種（A1 Mini、A1、P1S等）は全自動レベリングを搭載しており、操作は非常にシンプルです。
  </div>
  <div style={S.step}>
    <span style={S.num}>1.</span>
    <span style={{ color: "#f1f5f9" }}>タッチパネルまたはBambu Studioから「キャリブレーション」を選択</span>
  </div>
  <div style={S.step}>
    <span style={S.num}>2.</span>
    <span style={{ color: "#f1f5f9" }}>「ベッドレベリング」または「1クリックキャリブレーション」をタップ</span>
  </div>
  <div style={S.step}>
    <span style={S.num}>3.</span>
    <span style={{ color: "#f1f5f9" }}>プリンターが自動でノズルをベッド全体に動かし測定（3〜10分）</span>
  </div>
  <div style={S.step}>
    <span style={S.num}>4.</span>
    <span style={{ color: "#f1f5f9" }}>完了後はそのまま印刷開始できる</span>
  </div>
  <div style={S.tip}>
    <div style={{ color: "#06b6d4", fontWeight: "700", marginBottom: "0.5rem" }}>💡 いつ実行すべき？</div>
    <div style={{ ...S.p, marginBottom: 0 }}>
      Bambu Lab機種は印刷前に自動でレベリングチェックを行います。プリントシートを交換した時や、移動後の再設置後は手動でキャリブレーションを実行するのがおすすめです。
    </div>
  </div>
</section>

<section id="ender3" style={S.section}>
  <h2 style={S.h2}>Ender-3での手順（半自動）</h2>
  <div style={S.p}>
    Ender-3 V3 SEなどの半自動レベリング機種での手順です。
  </div>
  <div style={S.step}>
    <span style={S.num}>1.</span>
    <span style={{ color: "#f1f5f9" }}>メニューから「準備」→「オートホーミング」を実行（ノズルがX/Y/Z原点に戻る）</span>
  </div>
  <div style={S.step}>
    <span style={S.num}>2.</span>
    <span style={{ color: "#f1f5f9" }}>「自動ベッドレベリング」を実行（複数点を自動測定）</span>
  </div>
  <div style={S.step}>
    <span style={S.num}>3.</span>
    <span style={{ color: "#f1f5f9" }}>「Zオフセット調整」でノズルとベッドの隙間を微調整（次のセクション参照）</span>
  </div>
  <div style={S.step}>
    <span style={S.num}>4.</span>
    <span style={{ color: "#f1f5f9" }}>「設定を保存」を選択して完了</span>
  </div>
</section>

<section id="z-offset" style={S.section}>
  <h2 style={S.h2}>Zオフセット調整（紙を使う方法）</h2>
  <div style={S.p}>
    Zオフセットとは、ノズル先端とベッド表面の距離のことです。正確に調整することで第1層の密着が良くなります。
  </div>
  <div style={S.tip}>
    <div style={{ color: "#06b6d4", fontWeight: "700", marginBottom: "0.5rem" }}>📄 紙1枚を使った調整方法</div>
    <div style={{ ...S.p, marginBottom: "0.5rem" }}>
      <strong style={{ color: "#f1f5f9" }}>目標：</strong> A4コピー用紙1枚（約0.1mm）がノズルとベッドの間でわずかな抵抗を感じながら引き出せる状態
    </div>
    <div style={{ ...S.p, marginBottom: "0.5rem" }}>
      紙が引っかかって動かない → ノズルが近すぎ（Zオフセットを＋方向に微調整）
    </div>
    <div style={{ ...S.p, marginBottom: 0 }}>
      紙が全く抵抗なく動く → ノズルが遠すぎ（Zオフセットを－方向に微調整）
    </div>
  </div>
  <div style={S.p}>
    最終的には実際に印刷して第1層を目視確認するのが確実です。第1層が薄くベッドに潰れて密着していれば成功です。
  </div>
</section>

<section id="trouble" style={S.section}>
  <h2 style={S.h2}>うまくいかない場合</h2>
  <div style={{ overflowX: "auto" }}>
    <table style={{ width: "100%", borderCollapse: "collapse", fontSize: "0.875rem" }}>
      <thead>
        <tr>
          <th style={S.th}>症状</th>
          <th style={S.th}>原因と対処法</th>
        </tr>
      </thead>
      <tbody>
        <tr style={{ backgroundColor: "#0f172a" }}>
          <td style={{ ...S.td, color: "#ef4444" }}>フィラメントがベッドに付かない</td>
          <td style={S.td}>Zオフセットが大きすぎ → マイナス方向に調整 / ベッド温度を上げる</td>
        </tr>
        <tr>
          <td style={{ ...S.td, color: "#ef4444" }}>ノズルがベッドに当たる</td>
          <td style={S.td}>Zオフセットが小さすぎ → プラス方向に調整（印刷を即停止）</td>
        </tr>
        <tr style={{ backgroundColor: "#0f172a" }}>
          <td style={S.td}>一部の場所だけ密着が悪い</td>
          <td style={S.td}>ベッドの反りが疑われる → メッシュレベリングを使用、またはガラスベッドに交換</td>
        </tr>
        <tr>
          <td style={S.td}>毎回レベリングがズレる</td>
          <td style={S.td}>ベッドのネジが緩んでいる → 締め直してから再レベリング</td>
        </tr>
      </tbody>
    </table>
  </div>
</section>
