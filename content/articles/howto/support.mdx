<div style={S.card}>
  <div style={S.p}>
    <strong style={{ color: "#f1f5f9" }}>サポート材（支持材）は、空中に浮いた部分を支えるために一時的に出力される構造です。</strong> FDM方式の3Dプリンターは下から積み上げる方式のため、途中に空間がある形状ではサポートが必要になります。
  </div>
  <div style={{ ...S.p, marginBottom: 0 }}>
    サポートの種類・設定・取り外し方を理解すると、仕上がりが大きく向上します。
  </div>
</div>

<section id="why" style={S.section}>
  <h2 style={S.h2}>サポート材が必要な理由</h2>
  <div style={S.p}>
    FDM（熱溶解積層）方式は、フィラメントを下から順番に積み重ねて造形します。そのため、<strong style={{ color: "#f1f5f9" }}>下に何もない空間（オーバーハング）</strong>があると、フィラメントが空中に垂れてしまいます。
  </div>
  <div style={S.tip}>
    <div style={{ color: "#06b6d4", fontWeight: "700", marginBottom: "0.5rem" }}>📐 45度ルール</div>
    <div style={{ ...S.p, marginBottom: 0 }}>
      一般的に<strong style={{ color: "#f1f5f9" }}>45度以上のオーバーハング</strong>になるとサポートが必要です。45度以下であれば多くの場合サポートなしで印刷できます。ただしプリンターや素材によって異なります。
    </div>
  </div>
</section>

<section id="types" style={S.section}>
  <h2 style={S.h2}>サポートの種類</h2>

      </thead>
      <tbody>
        <tr style={{ backgroundColor: "#0f172a" }}>
          <td style={{ ...S.td, fontWeight: "700" }}>通常サポート（Normal）</td>
          <td style={S.td}>格子状の柱。シンプルで速いが、外し跡が残りやすい</td>
          <td style={S.td}>シンプルな形状・機能パーツ</td>
        </tr>
        <tr>
          <td style={{ ...S.td, color: "#06b6d4", fontWeight: "700" }}>ツリーサポート（Tree）</td>
          <td style={S.td}>枝状に伸びてオーバーハングを支える。外しやすく跡が残りにくい</td>
          <td style={{ ...S.td, color: "#06b6d4" }}>フィギュア・複雑な形状（初心者推奨）</td>
        </tr>
        <tr style={{ backgroundColor: "#0f172a" }}>
          <td style={S.td}>水溶性サポート（PVA）</td>
          <td style={S.td}>水に溶ける素材。完璧な仕上がりになるが、マルチノズル機が必要</td>

        </tr>
        <tr>
          <td style={S.td}>ブレークアウェイ</td>
          <td style={S.td}>本体と異なる素材でサポート。通常より外しやすい</td>
          <td style={S.td}>マルチカラー機・PETG＋PLA組み合わせ</td>
        </tr>
      </tbody>
    </table>
  </div>

    迷ったら<strong style={{ color: "#06b6d4" }}>ツリーサポート</strong>を選んでください。取り外しやすく仕上がりもきれいです。
  </div>
</section>

<section id="settings" style={S.section}>
  <h2 style={S.h2}>スライサーでの設定</h2>
  <div style={S.p}>
    <strong style={{ color: "#f1f5f9" }}>Bambu Studio の場合：</strong> プロセスタブ → 「サポート」セクションで「サポートを有効にする」をオン。タイプをNormal / Treeから選択。Z距離（ベッド面からの隙間）は0.2mmが標準です。
  </div>
  <div style={S.p}>
    <strong style={{ color: "#f1f5f9" }}>Cura の場合：</strong> 「サポートを生成」をオンにして、サポート構造を「通常」または「ツリー」から選択。オーバーハング角度は45度が標準設定です。
  </div>
  <div style={S.tip}>
    <div style={{ color: "#06b6d4", fontWeight: "700", marginBottom: "0.5rem" }}>💡 サポートのZ距離について</div>
    <div style={{ ...S.p, marginBottom: 0 }}>
      Z距離（サポートと本体の隙間）は取り外しやすさと面品質のバランスです。<strong style={{ color: "#f1f5f9" }}>0.2mm</strong>が一般的な推奨値。広げると外しやすくなりますが、面がザラつきます。
    </div>
  </div>
</section>

    </div>
  </div>
