<section id="bed" style={S.section}>
  <h2 style={S.h2}>ベッドレベリング</h2>
  <div style={S.p}>
    ベッドレベリングはノズルとベッド面の距離を均一にする作業です。ほとんどの現代的なプリンターは自動レベリング（ABL）を搭載しているため、定期的に実行するだけで対応できます。
  </div>
  <div style={S.tip}>
    <div style={{ color: "#06b6d4", fontWeight: "700", marginBottom: "0.5rem" }}>💡 自動レベリングの実行タイミング</div>
    <div style={{ ...S.p, marginBottom: 0 }}>
      プリンターを移動したとき / フィラメントを変えたとき / 1層目の定着が悪くなってきたとき / 月に1回程度の定期メンテナンス
    </div>
  </div>
  <div style={S.p}>
    手動レベリングの場合は、A4紙1枚分（約0.1mm）の隙間がノズルとベッドの間にある状態が目安です。4隅と中央の計5点を確認します。
  </div>
</section>

  </div>

      <tbody>
        <tr style={{ backgroundColor: "#0f172a" }}>
          <td style={{ ...S.td, color: "#ef4444" }}>フィラメントがベッドに全くつかない / 浮いている</td>
          <td style={S.td}>ノズルが高すぎ</td>
          <td style={S.td}>Zオフセットをマイナス方向に調整（ノズルを下げる）</td>
        </tr>
        <tr>
          <td style={{ ...S.td, color: "#22c55e" }}>ベッドにしっかり密着・ライン間に隙間なし</td>

          <td style={S.td}>ノズルが低すぎ</td>

      </tbody>

<section id="flow" style={S.section}>

  <div style={S.p}>

  </div>
  <div style={S.card}>

    <div style={{ ...S.p, marginBottom: "0.5rem" }}><strong style={{ color: "#f1f5f9" }}>Step 3</strong>：スライサーで設定したライン幅（例：0.4mm）と比較</div>
    <div style={{ ...S.p, marginBottom: 0 }}><strong style={{ color: "#f1f5f9" }}>Step 4</strong>：計測値が設定値より大きい場合はフローを下げ、小さければ上げる</div>
